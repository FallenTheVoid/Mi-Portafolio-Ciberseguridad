# SIGEP – Sistema Integral de Gestión de Empleados y Pagos

SIGEP es una aplicación web tipo SPA (Single Page Application) desarrollada como proyecto universitario para la gestión administrativa en pequeñas y medianas empresas. Permite al administrador realizar operaciones clave como:
- Registro y edición de empleados
- Asignación y visualización de turnos laborales
- Gestión de pagos y generación de recibos
- Creación de reportes personalizados
- Navegación fluida entre secciones sin recargar la página

**Tecnologías utilizadas**

- Frontend: HTML, CSS, JavaScript
- Backend: Python, Flask
- Base de datos: SQLite con SQLAlchemy

**Características destacadas**

- Panel administrativo con menú lateral y diseño responsive
- Modales interactivos para formularios y confirmaciones
- Tablas dinámicas con filtros y acciones
- API REST para comunicación entre frontend y backend
- Resúmenes visuales por sección (turnos, pagos, recibos)

Este proyecto fue desarrollado con fines académicos, aplicando buenas prácticas de diseño, estructura modular y experiencia de usuario.

**Cómo ejecutar el proyecto**

- Clona el repositorio:
git clone https://github.com/tu-usuario/sigep.git

- Instala dependencias del backend:
pip install flask sqlalchemy

- Ejecuta el servidor Flask:
python app.py

- Abre index.html con live server.
